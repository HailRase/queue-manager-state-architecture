## Вход в систему

Успешная авторизация: создание БД со всем таблицами, отправка запроса на получение всех данных для БД

## Вход с сессией
Создаётся ws соединение
Проверяем sid
Если sid валидный - проверка есть ли БД: нет - создаём и отправляем запрос на получение данных БД 
Если sid не валидный - переводим на страницу регистрации и чистим БД
